## ✨ Enterprise-Grade Features
- **Intelligent Ingestion**: Rapidly upload and parse heavy PDF documents through a pristine Streamlit user interface without sacrificing structural integrity.
- **Semantic Text Chunking**: Automatically fragments large documents into precise contextual blocks using LangChain's `RecursiveCharacterTextSplitter`, preserving paragraph flow and sentence-level context.
- **Local FAISS Vector Engine**: Secures your data by indexing and retrieving semantic information completely locally. Bypasses the need for costly external cloud vector databases.
- **OpenAI Integration**: 
  - Sub-millisecond embeddings powered by OpenAI's highly efficient `text-embedding-3-small`.
  - Contextually aware, hallucination-resistant answers generated by `gpt-3.5-turbo`.
- **Transparent Sourcing**: The assistant never guesses. Every answer is coupled with exact citations, snippets, and page metadata showing exactly where the information was extracted from visually within the interface.

## 🚀 Local Setup & Installation

### Prerequisites
- **Python 3.8+**
- **OpenAI API Key**: Funded and active. Generate yours from the [OpenAI Platform](https://platform.openai.com/api-keys).

### Installation Steps
1. **Clone or Navigate to the project directory**:
   ```bash
   cd path/to/your/project
   ```

2. **Create a secure virtual environment**:
   ```bash
   python -m venv venv
   
   # On Windows:
   venv\Scripts\activate
   
   # On macOS/Linux:
   source venv/bin/activate
   ```

3. **Install the ecosystem dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

4. **Inject your Credentials**:
   Create a `.env` file in the root directory and securely add your funded API key:
   ```env
   OPENAI_API_KEY="sk-your-funded-api-key-here"
   ```

### Execution
Boot up the Streamlit server locally:
```bash
streamlit run app.py
```
This will automatically launch the graphical interface at `http://localhost:8501`. Drop your PDF into the sidebar, initialize the processing, and begin querying!

## ⚙️ Configuration & Hyperparameters
Power users can tune the system's operational parameters inside `config.py`:
- **CHUNK_SIZE**: `1000` (The maximum tokens captured per vector embedding)
- **CHUNK_OVERLAP**: `200` (Preserved overlap space to stop sentences from being awkwardly split)
- **RETRIEVAL_K**: `4` (Number of document cluster matches utilized to shape the AI's final answer)
- **CHAT_MODEL**: Readily up-switchable to `gpt-4o-mini` or `gpt-4` for intense analytical reasoning.

*System note: The application manages filesystem hygiene automatically by creating `./uploads` and `./vector_store` repositories upon ingestion.*

## 🌐 Production Deployment Guide (Streamlit Cloud)
To showcase this project seamlessly on the web, leverage **Streamlit Community Cloud** (Free forever).

1. **Push to GitHub**:
   Upload this project folder to a repository. *The `.gitignore` strictly protects your `.env` key from becoming public!*

2. **Spin up your App**:
   - Access [share.streamlit.io](https://share.streamlit.io/) and authorize GitHub.
   - Click "**New app**", select your branch, and point the main file path to `app.py`.

3. **Secure Cloud Secrets (CRITICAL)**:
   - Before hitting deploy, tap "**Advanced settings...**".
   - Locate the **Secrets** section. Streamlit does not read `.env` files in production. Paste your configuration here identically:
     ```toml
     OPENAI_API_KEY="sk-your-actual-funded-api-key-here"
     ```
   - Click "**Save**".

4. **Launch!**
   Click "**Deploy!**". The cloud engine will provision the required dependencies and immediately serve your app globally via a public URL.
